# Layout 6 - Responsive Flex-Based User Interface

This project presents a **responsive layout** built using **HTML5** and **CSS3**, consisting of a profile/search header and an advanced, flex-based content layout with multiple nested components.

---

## Project Overview

- **Top Section**:

  - Avatar and search bar layout using **Flexbox**
  - Below it, three horizontally aligned **info cards** with square icons and text placeholders

- **Bottom Section**:
  - User detail section with avatar, nested boxes, and controls
  - Three vertically stacked boxes with custom widths and heights

- Designed to be fully **responsive** with fluid resizing and rearrangement of content

---

## Technologies Used

- HTML5
- CSS3
- Flexbox for structure
- CSS Custom Properties (variables)
- Media Queries for responsiveness

---

## What I Practiced

- Advanced **Flexbox** layout techniques with nested components
- Use of **CSS variables** for maintainability
- Responsive spacing, alignment, and adaptive resizing of containers
- Structuring content for clarity and reusability across devices

---

1. Clone the repository or download the ZIP
